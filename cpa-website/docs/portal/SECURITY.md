# Portal Security Notes

## Baseline Requirements
- MFA or passkeys for all accounts.
- Encryption in transit and at rest.
- Audit logging for document access.
- Malware scanning on uploads.
- Least-privilege roles for staff.
- AI chatbot: no PII accepted; redact logs; short retention for chat sessions.

## Open Questions
- Data retention period.
- Backup and disaster recovery targets.
- Key management ownership and rotation cadence.
- AI vendor data handling and retention guarantees.
